# Family × Tone — Optional Reference Shelf

A vocabulary library the Wedge offers to apprentices in Heat 2. **Not the commission grammar.** The commission is the **lens** derived from the Soul Brief; this shelf provides craftsmanship grounding an apprentice MAY draw from, MAY invent against, or MAY ignore.

Crossing the **family** axis (lineage — which design tradition) with the **tone** axis (intensity — how loud) yields archetypes an apprentice can use as shorthand when explaining their direction's craft. Tense pairings (e.g., *brutally-minimal Liquid*, *playful Brutalist*) often produce distinctive directions when the assigned lens demands them. But the lens leads. If no entry on this shelf serves the lens, invent. If the lens is best served by a vocabulary not yet named here, name it in your Direction Card.

> **Why this shelf is optional now (was the commission grammar before):** When Family × Tone *was* the commission grammar, three apprentices reading the same Opus and the same shelf converged to the same three slots across projects — the shelf has its own gravity (agency-portfolio aesthetics: Obys, Locomotive, Active Theory, Clay, MDS) and that gravity flattened product-soul into shelf-fits. The Soul Brief's lenses solve this at the source. The shelf survives as vocabulary, not as cage.

## Family axis (lineage)

Optional grounding when the lens calls for a lineage-anchored vocabulary.

- **Editorial / typographic** — Obys, Locomotive type, magazine-tradition, Swiss-poster.
- **Motion-driven / immersive** — Locomotive motion, Active Theory, Resn, Fantasy.
- **Refined craft / high-end** — Clay, Daniel Korpai, MDS, Tiempos-luxury.
- **Artistic / illustrative** — Zhenya Rynzhuk, Leo Natsume, hand-drawn maximalism.
- **Brutalist / raw** — Memphis, brutalist concrete, Constructivist propaganda.
- **Liquid / atmospheric** — Liquid Glass, gradient mesh worlds, Kuznetsov motion.
- **Organic / natural** — Art Nouveau line, Wabi-sabi, mid-century-modern warmth.

## Tone axis (intensity)

Optional grounding when the lens calls for a stated intensity register.

- **brutally minimal** — subtraction, silence, surgical typography, near-zero motion.
- **maximalist chaos** — layered everything, density, ornament, motion choreography.
- **retro-futuristic** — period-specific futurism (80s neon, 90s rave, Y2K chrome, cyberpunk).
- **organic / natural** — botanical, hand-touched, asymmetric warmth, soft edges.
- **luxury / refined** — high-end editorial, generous space, exquisite detail, restrained palette.
- **playful / toy-like** — bright, bouncy, friendly, ornamental for joy.
- **editorial / magazine** — typographic command, asymmetric grids, masthead presence.
- **brutalist / raw** — exposed structure, monospace tension, unrefined edges as feature.
- **art deco / geometric** — symmetry, ornament-with-rules, gold/black gravitas.
- **soft / pastel** — muted palette, low contrast, ambient calm.
- **industrial / utilitarian** — functional density, monospace data, no decorative weight.

## How to use this shelf (apprentice perspective)

1. Receive your **lens** from the Soul Brief (e.g., `instrument`, `weather`, `dwelling`, `archive`).
2. Read the Soul Brief's prose sections in full — *What it IS*, *What it ISN'T*, *Examples from life*, *Forbidden Defaults*.
3. Decide what the lens demands of typography, palette, motion, and spatial logic.
4. *Then*, optionally, ask: does any pairing on this shelf cleanly ground that craft? If yes, name it on the Direction Card. If no, invent vocabulary that does. If the question doesn't help, work without shelf reference — direct soul + lens is sufficient.
5. The Direction Card has an explicit **Vocabulary grounding** field: list the pairing if you used one, name the invented vocabulary if you reached past the shelf, or write "none — invented from soul + lens directly."
6. **Refuse the shelf's gravity.** If a Forbidden Default in the Soul Brief overlaps with a shelf entry that "would fit," the Forbidden Default wins. The shelf is a tool, not an authority.
